# api-starter-node-jwt-mongo
An starter repo for an API with JWT, made with Node, MongoDB &amp; Mongoose

### Features:
- Data validation
- i18n error messages (en_US & es_MX)
- Testing
- Babel ES6 => ES5
- Heroku deployment
- Seeder starter
- Custom error logs
- Sendgrid mailer
- JSON Web Tokens
- Forgot pwd email
- Forgot pwd auth
